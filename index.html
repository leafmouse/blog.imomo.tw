Apache Reverse Proxy 實作筆記

Apr 19th, 2013

以下是使用 Apache 實作 Reverse Proxy

  1. 編輯Apache設定檔

     vi /etc/httpd/conf/httpd.conf

  2. 檢查Apache是否安裝mod_proxy模組

    搜尋設定檔中是否有下列二行

     LoadModule proxy_module modules/mod_proxy.so
 LoadModule proxy_http_module modules/mod_proxy_http.so

  3. 拿掉開啟proxy功能

     ProxyRequests Off 

 <Proxy *> 
 Order deny,allow 
 Deny from all 
 Allow from all 
 </Proxy>

  4. ProxyPassReverse設定

     ProxyVia On 
 ProxyPass /redir http://another_ip_address/redir 
 ProxyPassReverse /redir http://another_ip_address/redir

Reference: http://www.wretch.cc/blog/siaocheng/2878629

發表Octopress文章的流程

Apr 18th, 2013

發表blog的流程

  1. 執行下列指令建立新的文章

     rake new_post["新文章標題"]

  2. 使用 Sublime Text 2 或其他編輯軟體編輯文章

  3. 執行下列指令以利於本機中先預覽 http://localhost:4000

     rake preview

  4. 編譯文章

     rake generate

  5. 將文章上傳到 github.com

     rake deploy

  6. 同時用git add, git commit, 和git push origin site 提交site分支的更改
