發表Octopress文章的流程

Apr 18th, 2013

發表blog的流程

  1. 執行下列指令建立新的文章

     rake new_post["新文章標題"]

  2. 使用 Sublime Text 2 或其他編輯軟體編輯文章

  3. 執行下列指令以利於本機中先預覽 http://localhost:4000

     rake preview

  4. 編譯文章

     rake generate

  5. 將文章上傳到 github.com

     rake deploy

  6. 同時用git add, git commit, 和git push origin site 提交site分支的更改
